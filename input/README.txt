1) DESCRIPTION

This is the README file for EmoEvent dataset

Emovent contains 8,409 annotate tweets written in Spanish. It is based on events that took place in April 2019 related to different domains: entertainment, catastrophe, political, global commemoration, and global strike.

The files included are: 

- train.tsv contains 5,886 annotated tweets. 
- dev.tsv contains 857 annotated tweets.

The dataset was annotated using crowdsourcing. The gold labels were assigned taking the agreement of three annotators into consideration.

Twitter user mentions were substituted by @USER and hasthags have been substitute by #HASHTAGS.

Each tweet is labeled with a unique emotion.

2) FORMAT

The column names in the file are the following:

id	tweet	label

The labels used in the annotation are listed below.

3) LABELS

   - anger (also includes annoyance and rage) 
   - disgust (also includes disinterest, dislike and loathing)
   - fear (also includes apprehension, anxiety, concern, and terror) 
   - joy (also includes serenity and ecstasy) 
   - sadness (also includes pensiveness and grief) 
   - surprise (also includes distraction and amazement)
